\# Design Reference Notes



\## Overall Direction



The website should feel like a premium white Liquid Glass SaaS tool.



The references are used for:

\- Layout quality

\- Spacing

\- Typography hierarchy

\- Card style

\- Button style

\- Mobile readability



Do not copy the exact content, branding, or colors from the references.



\## home-hero-reference.png



Use this reference for:

\- Hero layout

\- Large title hierarchy

\- Clear subtitle

\- Main CTA button placement

\- Generous white space

\- Premium first impression



Do not:

\- Copy the exact image content

\- Make it dark

\- Add unnecessary gaming effects



\## calculator-layout-reference.png



Use this reference for:

\- Left input panel

\- Right result panel

\- Professional calculator/settings layout

\- Clear grouped controls

\- Easy-to-read result cards



The calculator should feel like a real tool, not a random form.



\## vehicles-card-reference.png



Use this reference for:

\- Clean text-only cards

\- Directory/listing card hierarchy

\- Tags and metadata layout

\- Hover effect

\- Grid layout



Vehicle cards should work without images.



\## mobile-reference.png



Use this reference for:

\- Mobile card stacking

\- Mobile button size

\- Mobile spacing

\- Mobile typography

\- Touch-friendly inputs



Mobile should not feel like a squeezed desktop layout.



\## Final Quality Bar



The final UI should feel:

\- Clean

\- Modern

\- Premium

\- Readable

\- Lightweight

\- Trustworthy



It should not feel:

\- Cheap

\- Gray

\- Crowded

\- Like Bootstrap default

\- Like an admin panel

\- Like a dark gaming website
