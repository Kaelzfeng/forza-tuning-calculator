\# Forza Tuning Calculator Standalone Tool Site



\## Project Overview



This project is a standalone Forza tuning calculator website focused on:



\* suspension tuning

\* tire pressure tuning

\* gear ratio calculations

\* differential tuning

\* aero balancing

\* beginner-friendly tuning recommendations

\* mobile-first UI

\* SEO optimization

\* multilingual support



Tech stack:



\* Vue 3

\* Vite

\* TailwindCSS

\* Directus

\* Docker

\* Node.js
